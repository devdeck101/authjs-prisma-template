# Sobre o Projeto

O projeto está em constante manutenção e desenvolvimento.

![GitHub last commit](https://img.shields.io/github/last-commit/devdeck101/authjs-prisma-template)

![GitHub Repo stars](https://img.shields.io/github/stars/devdeck101/authjs-prisma-template)

Este é um template de projeto web que inclui autenticação e autorização implementado em [Next.js](https://nextjs.org/) utilizando [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

Foi incluído a inicialização com [Shadcn-ui](https://ui.shadcn.com/) para componentes de UI, [Prisma](https://www.prisma.io/) como camada de ORM e Authjs | Next-Auth [Authjs](https://authjs.dev/) como biblioteca de segurança e utiliza [PostgreSQL](https://www.postgresql.org/)
como base de dados

Além das functionalides básicas de segurança, está incluso neste Starter Kit functionalidade como:

| Nome | Tipo | Descrição |
| ---- | ---- | ----------- |
| `Multi-Step-Form` | Funcionalidade | Funcionalidade de processo guiado |
| `Website Builder` | Funcionalidade | Construção de paginas no-code |
